Установка StarredHeltix

Пошаговое руководство по установке мода

Системные требования

  • Minecraft Java Edition 1.21.4
  • Fabric Loader 0.17.2 или выше
  • Java 21 или выше
  • Не менее 4 ГБ оперативной памяти

1 Установка Fabric Loader

  1. Перейдите на официальный сайт FabricMC
  2. Выберите версию Minecraft 1.21.4
  3. Скачайте установщик Fabric для вашей операционной системы:
    • Windows: Скачайте executable jar
    • macOS/Linux: Скачайте universal jar
  4. Запустите скачанный файл и следуйте инструкциям установщика
  5. Установщик создаст профиль Fabric в вашем лаунчере Minecraft

2 Скачивание мода StarredHeltix

  1. Перейдите на страницу релизов
  2. Найдите последнюю версию мода
  3. Скачайте файл .jar (обычно называется starredheltix-x.x.x.jar)

3 Установка мода

  1. Откройте папку с Minecraft (обычно .minecraft)
    • Windows: Нажмите Win+R, введите %appdata%\.minecraft и нажмите Enter
    • macOS: Нажмите Cmd+Пробел, введите ~/Library/Application Support/minecraft
    • Linux: Папка находится в ~/.minecraft
  2. Создайте папку mods, если её ещё нет
  3. Скопируйте скачанный файл мода в папку mods

4 Запуск игры

  1. Откройте лаунчер Minecraft
  2. Выберите профиль Fabric (обычно называется "fabric-loader-1.21.4")
  3. Нажмите "Играть"
  4. После загрузки проверьте, что мод работает:
    • Введите команду /starredheltix в чат
    • Если мод установлен правильно, появится меню конфигурации

5 Устранение проблем

Если мод не работает:

  1. Проверьте, что вы используете правильную версию Minecraft (1.21.4)
  2. Убедитесь, что Fabric Loader установлен правильно
  3. Проверьте, что файл мода находится в папке .minecraft/mods
  4. Убедитесь, что у файла мода правильное расширение .jar
  5. Проверьте, что версия Java соответствует требованиям (Java 21+)

Если игра вылетает при запуске:

  1. Проверьте логи игры на наличие ошибок
  2. Убедитесь, что все зависимости установлены
  3. Попробуйте удалить конфигурационные файлы мода и перезапустить игру
  4. Проверьте, совместимы ли другие моды с StarredHeltix
