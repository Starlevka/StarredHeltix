Конфигурация StarredHeltix

Подробное руководство по настройке мода

Примечание: Все настройки можно изменить как через внутриигровые команды, так и через файлы конфигурации.

Основные команды

/starredheltix

Основная команда для доступа ко всем настройкам мода. Открывает главное меню конфигурации.

/starredheltix help

Отображает справку по всем доступным командам.

/starredheltix config <category>

Открывает настройки для определенной категории функций.

Категории настроек

Подсветка существ

Настройки для функций подсветки эндерменов, волков и персонажей.

Команда: /starredheltix config highlighter

  • enabled - Включить/выключить все функции подсветки (true)
  • enderman.enabled - Включить подсветку эндерменов (true)
  • wolf.enabled - Включить подсветку волков (true)
  • character.enabled - Включить подсветку персонажей (true)
  • duration - Продолжительность подсветки в тиках (100)
  • color - Цвет подсветки в формате RGB (#00FF00)
Автоспринт

Настройки функции автоматического спринта.

Команда: /starredheltix config autosprint

  • enabled - Включить автоспринт (true)
Снаряжение

Настройки отображения и обнаружения снаряжения.

Команда: /starredheltix config equipment

  • enabled - Включить отображение снаряжения (true)
  • slotLock.enabled - Включить блокировку слотов (true)
  • slotLock.slots - Список заблокированных слотов ([0, 1, 2, 3, 4])
Чат

Настройки функций чата.

Команда: /starredheltix config chat

  • partyCommands.enabled - Включить команды через групповой чат (true)
  • messageCopy.enabled - Включить копирование сообщений (true)
  • partySound.enabled - Включить звуковые уведомления группы (true)
Утилиты

Настройки различных утилит.

Команда: /starredheltix config utils

  • fairySouls.enabled - Включить точки маршрута Фейри Соулс (true)
  • experimentSolver.enabled - Включить решатель экспериментального стола (true)
  • bloodRoomTimer.enabled - Включить таймер кровавой комнаты (true)
  • fullInventoryWarning.enabled - Включить предупреждение о полном инвентаре (true)

Файлы конфигурации

Конфигурационные файлы хранятся в папке .minecraft/config/starredheltix/:

  • general.json - Основные настройки мода
  • highlighters.json - Настройки подсветки существ
  • equipment.json - Настройки снаряжения
  • chat.json - Настройки чата
  • utils.json - Настройки утилит

Файлы создаются автоматически при первом запуске мода и могут быть изменены вручную.
