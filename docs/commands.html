Команды StarredHeltix

Полный список команд мода и их использование

Примечание: Все команды начинаются с символа /. Некоторые команды требуют определенных прав для использования.

Основные команды

/starredheltix Все игроки

/starredheltix [help|config|reload]

Основная команда мода. Без параметров открывает главное меню конфигурации.

  • help - Отображает справку по всем командам
  • config - Открывает меню конфигурации
  • reload - Перезагружает конфигурационные файлы

/вход Все игроки

/вход

Команда для автоматического входа в аккаунт. Может использоваться для быстрого подключения к серверу.

/яготовлёвал Все игроки

/яготовлёвал

Отправляет сообщение "Я готовлёвал!" в групповой чат. Полезно для координации с другими игроками.

Команды группы

Эти команды могут быть выполнены через групповой чат, начиная сообщение с префикса !.

!promote

!promote <ник игрока>

Повышает указанного игрока в группе.

!demote

!demote <ник игрока>

Понижает указанного игрока в группе.

!kick

!kick <ник игрока> [причина]

Выгоняет указанного игрока из группы с необязательной причиной.

!invite

!invite <ник игрока>

Приглашает указанного игрока в группу.

!transfer

!transfer <ник игрока>

Передает лидерство в группе указанному игроку.

!warp

!warp

Телепортирует всех участников группы к лидеру.

Команды личных сообщений

Эти команды могут быть выполнены через личные сообщения, начиная сообщение с префикса !.

!ah

!ah <ник игрока>

Открывает меню аукциона указанного игрока.

!skills

!skills <ник игрока>

Отображает навыки указанного игрока.

!bank

!bank <ник игрока>

Отображает банковский баланс указанного игрока.

Команды снаряжения

/equipment Все игроки

/equipment

Открывает меню управления снаряжением. Также может быть вызвано кликом по слотам снаряжения в инвентаре.

Команды утилит

/fs Все игроки

/fs [near]

Управление точками маршрута Фейри Соулс. Без параметров показывает все точки, с параметром near показывает ближайшие точки.

/solver Все игроки

/solver <experiment|chronomatron|ultrasequencer>

Открывает решатели головоломок для различных экспериментов:

  • experiment - Решатель экспериментального стола
  • chronomatron - Решатель Хрономатрона
  • ultrasequencer - Решатель Ультрасеквенсора
